Сборка образа производится в 2 этапа.
На первом этапе производится сборка образа с языком програмирования golang.
В этом образе компилируется приложения.

На втором этапе производится сборка образа на основе ОС Debian.
Сюда копируется скомпилированное приложения с предыдущего образа.

Для выполнения сборки итогового образа вводим команду "sudo docker build -t example-build ."
В итоге собирается образ example-build с готовым приложением.

Командой "sudo docker run --rm --name example -d -p 3535:3535 example-build" запускаем контейнер на основе образа example-build.
После запуска контейнера, у приложения внутренний порт 3535, внешний порт 3535.
